# Codex Prompt Recipes

Codex does not need Claude-style slash command files. These recipes preserve the original ECC command workflows as pasteable prompts.

Recommended usage:

```text
Use the code-review prompt recipe on my current git diff.
Use the tdd-workflow skill and follow RED -> GREEN -> REFACTOR for this feature.
```

For reusable knowledge, prefer installing the skills in `skills/.curated` into `$CODEX_HOME/skills`.
